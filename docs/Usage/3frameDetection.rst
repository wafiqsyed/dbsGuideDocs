Frame Detection
==================

.. image:: ../Images/widget03.png
   :align: center

1. Select CT Frame
	Now it's time to set up the frame fiducial points of our CT frame. Click on the drop down menu
	next to ``Fiducial Volume`` and select ``CT Frame``. Then click  ``Confirm Frame Fiducials``.
